  2. #ID:ID选择器

    就是用#索引。ID选择器是CSS中效率最高的选择器，使用的时候注意保证ID的唯一性。

  4. X，Y：多元素选择器

    同时匹配所有X或Y元素，用，隔开

  5. X Y: 后代元素选择器

    匹配所有X元素后代的Y元素，用空格分隔

  6. X>Y： 子元素选择器

    匹配X元素的所有Y元素子元素

    css选择器中后代选择器与子选择器的区别
    后代选择器：就是对象当中所有的子元素，孙子元素以及最底层的元素

    子选择器：就单单是子元素，子元素的子元素就不算了

  7. X+Y:匹配所有紧随X元素之后的同级元素Y

  8. E[att]:匹配所有具有att属性的E元素，不考虑它的值。（注意：E在此处可以省略，比如“[cheacked]”。以下属性选择器同。）

  9. E[att=val],匹配所有att属性等于“val”的E元素.

  10. E[att~=val],匹配所有att属性具有多个空格分隔的值、其中一个值等于“val”的E元素

  11. E[att|=val],匹配所有att属性具有多个连字号分隔（hyphen-separated）的值、其中一个值以“val”开头的E元素，主要用于lang属性，比如“en”、“en-us”、“en-gb”等等.

    CSS 2.1 属性选择器还有一个特点就是使用多个选择器，同时满足这多个选择器：blockquote[class=quote][cite] { color:#f00; }

    下面是CSS 2.1 中的伪类：(后面有几个示例)

  12. E:first-child，匹配父元素的第一个子元素。

  13. E:link,匹配所有未被点击的链接。

  14. E:visited，匹配所有已被点击的链接。

  15. E:active，匹配鼠标已经其上按下、还没有释放的E元素。

  16. E:hover，匹配鼠标悬停其上的E元素。

  17. E:focus，匹配获得当前焦点的E元素。

  18. E:lang(c)，匹配lang属性等于c的E元素。

    下面是CSS 2.1 中的伪元素：(后面有几个示例)

  19. E:first-line,匹配E元素的第一行。

  20. E:first-letter,匹配E元素的第一个字母。

  21. E:before,在E元素之前插入生成的内容。

  22. E:after，在E元素之后插入生成的内容。

    CSS 3的同级元素通用选择器

  23. E ~ F，匹配任何在E元素之后的同级F元素。

    CSS 3 属性选择器

  24. E[att^=”val”],属性att的值以”val”开头的元素。

  25. E[att$=”val”],属性att的值以”val”结尾的元素。

  26. E[att*=”val”]，属性att的值包含”val”字符串的元素。

    CSS 3中与用户界面有关的伪类

  27. E:enabled，匹配表单中激活的元素。

  28. E:disabled,匹配表单中禁用的元素。

  29. E:checked,匹配表单中被选中的radio（单选框）或checkbox（复选框）元素。

  30. E::selection，匹配用户当前选中的元素。

    CSS 3中的结构性伪类

  31. E:root，匹配文档的根元素，对于HTML文档，就是HTML元素。

  32. E:nth-child(n)，匹配其父元素的第n个子元素，第一个编号为1。

  33. E:nth-last-child(n),匹配其父元素的倒数第n个子元素，第一个编号为1。

  34. E:nth-of-type(n),与:nth-child()作用类似，但是仅匹配使用同种标签的元素。

  35. E:nth-last-of-type(n),与:nth-last-child() 作用类似，但是仅匹配使用同种标签的元素。

  36. E:last-child,匹配父元素的最后一个子元素，等同于:nth-last-child(1)。

  37. E:first-of-type,匹配父元素下使用同种标签的第一个子元素，等同于:nth-of-type(1)。

  38. E:last-of-type,匹配父元素下使用同种标签的最后一个子元素，等同于:nth-last-of-type(1)。

  39. E:only-child,匹配父元素下仅有的一个子元素，等同于:first-child:last-child或 :nth-child(1):nth-last-child(1)。

  40. E:only-of-type,匹配父元素下使用同种标签的唯一一个子元素，等同于:first-of-type:last-of-type或。

    :nth-of-type(1):nth-last-of-type(1)。

  41. E:empty,匹配一个不包含任何子元素的元素，注意，文本节点也被看作子元素。

    CSS 3的反选伪类

  42. E:not(s),匹配不符合当前选择器的任何元素。

    CSS 3中的 :target 伪类

  43. E:target,匹配文档中特定”id”点击后的效果。
